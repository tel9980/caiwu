## 🚀 快速开始

### Q1: 第一次使用应该怎么做？
**A:** 按以下步骤操作：
1. 打开Excel模板文件
2. 点击【首页导航】工作表
3. 点击【初始化系统】按钮（或运行宏 `InitSimpleSystem`）
4. 在【基础设置】中修改客户、供应商信息
5. 开始录入数据！

### Q2: 如何启用宏功能？
**A:** 
- **Excel 2010及以上**：文件 → 选项 → 信任中心 → 信任中心设置 → 宏设置 → 启用所有宏
- **WPS表格**：开发工具 → 宏安全性 → 调低安全级别
- **提示**：启用宏后需要重新打开文件

### Q3: 点击按钮没有反应？
**A:** 检查以下几点：
1. 是否启用了宏？（见Q2）
2. 文件是否为`.xlsm`格式？（如果是`.xlsx`，宏会被禁用）
3. 是否将VBA代码导入到文件中？

## 💰 收入录入

### Q4: 收入什么时候录入？
**A:** 
- **推荐**：月底与客户对账后，统一批量导入
- **格式**：日期,客户,金额,收款方式,备注
- **示例**：`2025-06-15,永达五金厂,15000,G银行转账,5月加工费`

### Q5: 可以录入多笔收入吗？
**A:** 可以！使用【导入对账收入】功能，一次粘贴多行：
```
2025-06-05,永达五金厂,25000,G银行转账,5月加工费
2025-06-08,恒丰铝业,18000,G银行转账,5月加工费
2025-06-12,鑫盛金属,32000,G银行转账,5月加工费
```

### Q6: 收入录入错误如何修改？
**A:** 直接在【收入记录】工作表中找到对应行，修改单元格内容即可。

## 💸 支出录入

### Q7: 支出有哪些类别？
**A:** 共12个类别：
| 编号 | 类别 | 常见用途 |
|-----|------|---------|
| 1 | 房租 | 厂房租金 |
| 2 | 水费 | 生产用水 |
| 3 | 电费 | 生产用电 |
| 4 | 三酸 | 硫酸、硝酸、盐酸 |
| 5 | 片碱 | 氢氧化钠 |
| 6 | 亚钠 | 亚硝酸钠 |
| 7 | 色粉 | 染色材料 |
| 8 | 除油剂 | 前处理材料 |
| 9 | 挂具 | 工装夹具 |
| 10 | 外发加工费 | 委外加工 |
| 11 | 日常费用 | 办公、交通 |
| 12 | 工资 | 员工工资 |

### Q8: 月底如何快速录入多笔支出？
**A:** 使用【批量录入支出】功能：
1. 运行宏 `BatchAddExpense`
2. 按格式粘贴多行数据：
```
2025-06-05,房租,5000,房东,G银行转账,6月房租
2025-06-10,电费,3500,供电局,G银行转账,5月电费
2025-06-15,工资,15000,,G银行转账,6月工资
```

### Q9: 外发加工费如何录入？
**A:** 使用【录入外发加工费】功能：
- 格式：日期,厂家,加工内容,数量,单价,付款情况,备注
- 示例：`2025-06-15,XX电镀厂,镀金加工,100,5,已付,客户急单`
- **注意**：外发加工费会自动同步到支出记录！

## 📊 报表生成

### Q10: 如何生成利润表？
**A:** 三种方式：
1. 点击【首页导航】→【生成利润表】
2. 运行宏 `GenerateSimpleReport`
3. 查看【利润表】工作表（自动计算）

### Q11: 利润表数据不对？
**A:** 检查以下几点：
1. 【收入记录】是否有数据？
2. 【支出记录】是否有数据？
3. 【外发加工费】是否与支出表一致？
4. 运行【数据校验】检查问题

### Q12: 月底需要做哪些操作？
**A:** 推荐流程：
```
导入收入 → 导入应收应付 → 数据校验 → 一键月结 → 查看仪表盘 → 加密备份
```
或直接使用【一键月结】自动完成！

## 📥 数据导入

### Q13: 如何导入客户对账单？
**A:** 
1. 从客户处获取对账数据（Excel或文本）
2. 运行【导入对账收入】或【Excel文件导入】
3. 按提示粘贴数据或选择文件

### Q14: 微信/支付宝账单如何导入？
**A:**
1. **微信导出**：我 → 服务 → 钱包 → 账单 → 常见问题 → 下载账单
2. **支付宝导出**：我的 → 账单 → ... → 开具交易流水证明
3. 运行【微信/支付宝导入】，选择CSV文件
4. 系统会自动识别并分类

### Q15: 导入的支出类别不对？
**A:** 
- 微信/支付宝导入后，请在【支出记录】中检查分类
- 可手动修改类别列
- 系统会根据交易对方和备注智能分类，但可能不完全准确

## 🔒 数据安全

### Q16: 如何备份数据？
**A:** 两种方式：
1. 【数据备份】：普通备份到`\备份\`文件夹
2. 【加密备份】：带密码的加密备份（更安全）

### Q17: 忘记密码怎么办？
**A:** ⚠️ **重要提示**：
- 密码无法恢复！
- 请妥善保管密码，建议记录在安全的地方
- 定期做无密码备份，以防万一

### Q18: 如何设置文件密码？
**A:** 
1. 运行【设置密码保护】
2. 输入密码（至少6位）
3. 再次确认密码
4. 保存文件

## 🤖 AI功能

### Q19: AI异常检测是什么？
**A:** 自动识别以下异常：
- 单日收入远超平均值
- 某类别支出比上月突增50%以上
- 经营亏损月份
- 大额应收（>8万）

**建议**：每月运行一次

### Q20: 趋势预测需要多少数据？
**A:** 需要至少3个月的历史数据才能进行预测。

### Q21: 智能经营建议准确吗？
**A:** 基于您的实际数据生成，包括：
- 盈利能力分析
- 成本结构优化建议
- 资金管理建议
- 税务优化提醒

建议结合实际情况参考。

## 👥 多用户协作

### Q22: 如何设置多用户？
**A:**
1. 运行【用户权限设置】
2. 输入用户名
3. 选择角色（管理员/会计/出纳/查看者）
4. 在【基础设置】中查看用户列表

### Q23: 不同角色有什么区别？
**A:**
| 角色 | 权限 |
|-----|------|
| 管理员 | 全部功能，可设置其他用户权限 |
| 会计 | 可录入收入/支出，可生成报表，可执行月结 |
| 出纳 | 可录入支出，可查看报表，不可录入收入 |
| 查看者 | 仅可查看数据，不可录入或修改 |

### Q24: 如何记录操作日志？
**A:**
1. 运行【记录操作日志】
2. 输入操作描述
3. 输入操作人姓名
4. 使用【查看操作日志】查看记录

## 🐛 常见问题

### Q25: 公式显示#REF!错误？
**A:** 
- 可能是引用的单元格被删除
- 检查【收入记录】【支出记录】是否存在
- 重新运行【初始化系统】修复

### Q26: 报表数字为0？
**A:**
- 检查源数据工作表是否有数据
- 确认数据行号是否正确（从第4行开始）
- 运行【数据校验】检查

### Q27: 如何修改客户/供应商名称？
**A:**
- 在【基础设置】工作表中修改
- 注意：修改后历史数据不会自动更新

### Q28: 可以添加新的支出类别吗？
**A:** 可以！在【支出记录】中直接输入新类别名称，系统会自动识别。

### Q29: 应收应付如何导入？
**A:** 使用【导入应收应付余额】：
- **应收格式**：客户名,期初,本期增加,本期收款,期末
- **应付格式**：应付,供应商名,期初,本期增加,本期付款,期末
- 系统会自动进行勾稽校验

### Q30: 勾稽校验失败怎么办？
**A:** 
- 检查公式：期初 + 本期增加 - 本期收款 = 期末
- 确认数据是否准确
- 可选择忽略继续导入，但建议核对数据
