# 小型氧化加工厂财务管理系统 V18.0 Python版

## 🚀 版本特点

**Python独立运行版** - 无需WPS/Excel VBA环境！

- ✅ 纯Python实现，跨平台运行（Windows/Mac/Linux）
- ✅ 数据保存为标准Excel格式，兼容WPS/Excel
- ✅ **彩色终端界面** - 仪表盘、两列菜单、表格输出
- ✅ **45个完整功能** - 覆盖记账/报表/税务/工资/预算/提醒
- ✅ **智能提醒系统** - 应收逾期/库存预警/利润预警
- ✅ **七级累进个税** - 3%-45%税率自动计算
- ✅ **动态模拟数据** - 自动生成近3个月业务数据

## 📦 快速开始

### 方法一：双击启动（Windows）

1. 双击 `启动财务系统.bat`
2. 首次运行自动安装依赖（openpyxl）
3. 按菜单编号选择功能，按回车查看仪表盘

### 方法二：命令行启动

```bash
# 安装依赖
pip install openpyxl

# 运行程序
python 氧化加工厂财务系统.py
```

## ❓ 常见问题

### Q: 提示"缺少openpyxl"？
```bash
pip install openpyxl
```

### Q: Windows下没有颜色显示？
- Windows 10/11 默认支持ANSI颜色
- 如显示异常，尝试在PowerShell中运行

### Q: 数据保存在哪里？
数据保存在程序目录下的 `财务数据/财务数据.xlsx`

### Q: 如何备份数据？
- 方法1：菜单选择 [32] 自动备份
- 方法2：直接复制 `财务数据.xlsx` 文件
- 方法3：备份整个 `财务数据/` 目录

### Q: 如何在其他电脑使用？
1. 复制整个文件夹
2. 安装Python 3.8+
3. `pip install openpyxl`
4. 运行 `python 氧化加工厂财务系统.py`

### Q: 能否与VBA版共用数据？
可以！两个版本使用相同的Excel文件格式，数据完全兼容。
