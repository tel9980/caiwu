# 小型氧化加工厂财务管理系统

专为**小型五金氧化加工厂**设计的完整财务管理系统，提供 **Excel VBA版** 和 **Python独立版** 两种选择。

---

## 一、系统选择

| 特性 | Excel VBA版 | Python独立版 |
|------|-------------|-------------|
| 运行环境 | WPS/Excel（需启用宏） | Python 3.8+ |
| 安装 | 导入VBA模块 | `pip install openpyxl` |
| 跨平台 | 仅Windows | Windows/Mac/Linux |
| 界面 | Excel对话框 | 彩色终端界面 |
| 功能数量 | 52个宏 | 45个功能 |
| 数据格式 | Excel | Excel（完全兼容） |
| 版本 | V24.0 | V18.0 界面优化版 |

**数据完全互通**，可在两个版本间无缝切换。

---

## 二、Python版 (V18.0 界面优化版)

### 快速开始

```bash
pip install openpyxl
python 氧化加工厂财务系统.py
```

或双击 `启动财务系统.bat`（Windows）。

## 三、Excel VBA版 (V24.0)

### 快速入门

1. 打开 `氧化加工厂管理系统V18.xlsx`
2. 启用宏（Excel：信任中心→宏设置→启用所有宏）
3. 运行宏 `OneKeyInit` 初始化
4. 开始使用

### V24.0 核心亮点

- **超级快捷录入** - 订单录入效率提升6倍
- **批量收款录入** - 20笔从15分钟→2分钟
- **智能对账助手** - 自动匹配银行流水
- **每日经营简报** - 一键查看经营概况
- **微信收款识别** - 粘贴截图文字自动录入

### 快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl+Shift+O | 超级快捷录入订单 |
| Ctrl+Shift+R | 批量收款录入 |

## 五、注意事项

1. **VBA版**：先导入`VBA宏代码_V18.0_界面优化版.bas`，再启用宏
2. **Python版**：首次运行自动安装依赖，数据保存在 `财务数据/财务数据.xlsx`
3. **数据备份**：建议定期备份 `财务数据/` 目录
4. **首次使用**：务必先运行初始化功能
5. **会计准则**：《小企业会计准则》（财会〔2011〕17号）
