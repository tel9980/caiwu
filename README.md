# 氧化加工厂管理系统 V24.0

## 📋 系统简介

专为**小型五金氧化加工厂**设计的完整财务管理系统，覆盖从订单录入到纳税申报的全流程。

**V24.0核心亮点**：
- 🚀 **日常实务优化** - 订单录入效率提升6倍，批量收款一键完成
- 🤖 **智能自动化** - 凭证自动生成、报表自动取数、税务自动计算
- 📱 **移动适配** - 微信收款截图识别、语音输入格式支持
- ✅ **一键报税** - 利润表/资产负债表/纳税申报表一键生成

## ✨ V24.0 新增功能（日常实务优化）

### 1. 超级快捷录入 - 订单

**解决痛点**：录入一个订单要弹6个窗口，太麻烦

**优化效果**：
| 优化项 | 优化前 | 优化后 |
|-------|-------|-------|
| 操作步骤 | 6个弹窗分步输入 | 1个智能表单 |
| 输入方式 | 逐个字段填写 | 逗号分隔一次性粘贴 |
| 客户选择 | 手动输入 | 智能匹配+模糊查找 |
| 记忆功能 | 无 | 自动记忆上次客户/产品 |

**使用方式**：
```
按 Ctrl+Shift+O
↓
输入：2025-06-15,永达五金厂,铝件氧化-银白,按件,100,2.5
↓
系统自动识别日期、客户、产品、计价方式、数量、单价
```

**智能特性**：
- ✅ 客户名模糊匹配（输入"永达"→自动匹配"永达五金厂"）
- ✅ 计价方式智能转换（"1"/"件"/"piece"→"按件"）
- ✅ 日期错误自动纠正
- ✅ 自动记忆上次录入内容

---

### 2. 批量收款录入

**解决痛点**：月底集中收款时逐笔录入太慢

**使用方式**：
```
按 Ctrl+Shift+R
↓
批量粘贴（每行一笔）：
2025-06-15,永达五金厂,5000,G银行转账,DD2025061501
2025-06-15,恒丰铝业,3200,微信收款
2025-06-16,鑫达精密,8800,N银行转账
↓
系统自动录入并核销关联订单
```

**效率提升**：20笔收款从15分钟→2分钟

---

### 3. 智能对账助手

**解决痛点**：银行流水和业务单据对账费时费力

**功能**：
- 自动匹配银行流水和收款记录
- 金额+客户名相似度双重匹配
- 生成对账结果表，区分已匹配/未匹配

**使用方式**：
```
按 Ctrl+Shift+M
↓
系统自动分析银行流水
↓
输出对账结果：
  ✓ 已匹配：38笔（金额+客户名匹配）
  ? 未匹配：5笔（需人工核对）
```

### 5. 微信收款快速录入

**解决痛点**：微信收款后手动录入麻烦

**支持3种输入方式**：

**方式1：微信截图文字粘贴**
```
粘贴：当前状态 朋友已确认收钱 ￥2,500.00 永达五金厂 今天 08:30
系统自动提取：金额2500元，客户永达五金厂，日期今天
```

**方式2：语音输入格式**
```
输入：今天收永达五金厂2500
系统自动识别
```

**方式3：简单格式**
```
输入：永达五金厂,2500
```

## ⚠️ 注意事项

1. **导入顺序**：先导入`VBA宏代码_公共模块.bas`，再导入其他模块
2. **数据备份**：建议每周运行一次`BackupData`宏备份数据
3. **期初设置**：首次使用务必运行`OneKeyInit`进行初始化
4. **科目编码**：系统使用《小企业会计准则》科目编码
5. **税率设置**：小规模纳税人3%增值税，小微企业所得税分段计算

## 📞 技术支持

**常见问题**：
- Q: 客户名称输错了怎么办？  
  A: 系统会自动提示相似客户名，或先去【基础设置】添加新客户

- Q: 订单录错了能修改吗？  
  A: 直接在【加工订单】表中修改对应行

- Q: 月底报表数据不对？  
  A: 先运行`TrialBalanceCheck`检查凭证是否平衡

- Q: 银行对账对不上？  
  A: 使用`SmartReconcile`智能对账，查看未匹配项
